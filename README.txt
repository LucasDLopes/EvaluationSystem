Project requires:
XAAMP
SAP crystal reports
to be installed on machine.

please find included in this repo a SQL file.

Initialize XAAMP's apache and MySQL services
navigate to: http://localhost/phpmyadmin/index.php
Create a new database, name it: mock_university
Select the database and click on import tab
within import tab, navigate to the SQL file mentioned previously and use it to import.

Be sure to have SAP crystal reports installed on the machine. (or otherwise DO NOT CLICK the reports tab)
